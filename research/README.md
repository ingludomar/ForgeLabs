# Research — SyncBridge

Área de investigación del ecosistema. Aquí se registran ideas, experimentos, análisis y hallazgos que podrían evolucionar en features, mejoras de arquitectura o decisiones técnicas.

---

## Cómo usar esta área

Cada archivo representa un proyecto o el área general. Un item de investigación **no es un PROMPT ni una tarea** — es una hipótesis o exploración. Cuando una investigación concluye con una decisión de implementar, se convierte en PROMPT o feature.

### Estados posibles

| Estado | Descripción |
|---|---|
| 💡 Idea | Observación inicial — sin explorar |
| 🔍 Investigando | En proceso de análisis |
| ✅ Concluida | Decisión tomada — puede derivar en PROMPT/feature |
| 🚫 Descartada | Analizada y descartada con justificación |
